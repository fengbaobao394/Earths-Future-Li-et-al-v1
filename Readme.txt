This folder contains data in the paper for which the websites cannot be accessed because of access permission. 
The folder named Statistical Yearbook of Xinjiang contains the proportion of industrial GDP to total GDP. 
The folder named Sub-basin boundaries contains  the sub-basin boundaries in Xinjiang. 
The folder named Water Resources Bulletin of Xinjiang contains the water demand per 10000 GDP,  domestic water demand per capita, and ecological water demand in prefecture-level cities.
